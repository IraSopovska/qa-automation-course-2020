1.Verify the user has the ability to add the item from the Sports Tap into the basket (cart)

Steps:

1. Go to https://www.ebay.com/
2. Click on Sports Tab
3. Click on Cycling category on the left menu
4. Click on the Bicycle sub category on the left menu
5. Select Mountain Bike type
6. Do not sort ( Best Match is selected by default)
7. Select the first bike from the list ( Mountain Bike/Bicycle NEW SPEED Fat Tire)
8. Press 'Add to cart' button
9. Verify the selected bicycle is added to the basket

Expected result: The selected bicycle should be added to the basket

3. Press 'Search' button next to Search field
4. Verify the list with all Smart Watches are displaying below the Search field

3. Verify the user has the ability to be redirected to eBay Site of another country (Poland)

Steps:

1. Go to https://www.ebay.com/
2. Scroll to the bottom of the page
3. Find the eBay Sites label in the lower right corner of the page
4. Select Poland from the list
5. Verify the user is redirected to the Poland eBay site ( https://www.ebay.pl/)

Expected result: The user should be redirected to the Poland eBay site ( https://www.ebay.pl/)
